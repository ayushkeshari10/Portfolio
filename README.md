# Ayush Keshari | Advanced Interactive Portfolio

Welcome to the source code of my personal portfolio. This website is built to push the limits of pure HTML, CSS, and vanilla JavaScript—delivering an Awwwards-tier user experience without relying on heavy frontend frameworks or WebGL libraries.

## 🚀 Key Features & Visual Effects

- **Geometric Shutter Reveal:** A brutalist, highly-advanced preloader sequence that uses CSS polygon `clip-path` masks to physically slice the screen into vertical columns, ripping apart the glowing neon typography to seamlessly reveal the website.
- **Physics-Based Typography:** A custom JavaScript physics engine that splits text into individual DOM nodes, allowing letters to physically react, dodge, and tilt in 3D space when the cursor approaches them.
- **Holographic Liquid Gradients:** Premium, animated gradient sheens applied to text and backgrounds to simulate physical crystalline lighting.
- **3D Magnetic Cards:** Project and Education cards that track cursor movement and tilt dynamically using `rotateX` and `rotateY` transforms, complete with dynamic lighting glints.
- **Glassmorphic UI:** Modern frosted-glass aesthetics with `backdrop-filter` blurring for a sleek, premium dark-mode presentation.
- **Floating Label Forms:** A responsive contact grid featuring CSS-only floating labels and neon focus-state glows.

## 🛠️ Technology Stack
- **HTML5**: Semantic structure and SVG integrations.
- **CSS3**: Advanced keyframes, `clip-path` masking, 3D perspectives, CSS Variables, and Flexbox/Grid layouts.
- **Vanilla JavaScript (ES6+)**: Custom physics calculations, DOM manipulation, and precise `requestAnimationFrame` timing sequences.

## 💻 Running Locally
To view this project locally, simply clone the repository and open the HTML file:
```bash
git clone https://github.com/ayushkeshari10/Portfolio.git
cd Portfolio
```
Then, double-click on `index.html` or run it through a local live server (like VS Code's Live Server extension).
